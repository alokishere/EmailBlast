✦ Core Features
📝
Rich Compose Editor
Write plain text or full HTML emails. Supports template variables like {name} for personalization.
👥
Bulk Recipients
Paste a comma-separated list of hundreds of emails instantly. Tag-based UI keeps it clean.
📎
File Attachments
Drag & drop any file — PDFs, images, Word docs, ZIPs. Sent with every email in the batch.
🔐
Google OAuth
Emails send directly from your Gmail via OAuth. No passwords stored. No third-party relay.
⚡
Instant Delivery
Sends all emails in parallel. No queue delays or sending limits imposed by the app itself.
📊
Live Status
Real-time feedback shows how many sent successfully, how many failed, and which ones.
✦ How It Works
01
Sign in with Google
Authenticate securely via OAuth 2.0. MailBlast never stores your password or reads your inbox.
02
Compose Your Email
Write your subject, plain text body, and optionally an HTML version. One message, sent to all.
03
Add Recipients
Type or paste emails separated by commas. Invalid emails are flagged before sending.
04
Attach Files (Optional)
Drop any files onto the upload zone. All recipients receive the same attachments.
05
Hit Send. Done.
MailBlast fires off all emails through your Gmail account and shows you a delivery report.
✦ Honest Overview
✓  What's Great
  • ▸ Completely free, no subscription

  • ▸ Supports file attachments of any format

✗  Limitations
  • ▸ Bound by Gmail's daily send limits (~500/day free)
  • ▸ No email scheduling (yet)
  • ▸ No open/click tracking
  • ▸ Requires Google account to use
  • ▸ No contact list management
  • ▸ Single sender per session
